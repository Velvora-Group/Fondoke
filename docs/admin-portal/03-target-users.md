## 3.1 Regular User (Leisure Traveler)

**Profile**: Sarah, 32, Marketing Manager  
**Goals**:

- Find best hotel deals
- Flexibility to resell if plans change
- Easy booking process

**Pain Points**:

- Lost money on non-refundable bookings when plans change
- Limited options when searching across multiple platforms
- Difficulty finding last-minute deals

## 3.2 Advanced User (Frequent Traveler / Reseller)

**Profile**: Michael, 28, Travel Blogger  
**Goals**:

- Buy discounted reservations
- Profit from reselling hotel bookings
- Access to exclusive deals

**Pain Points**:

- Want to monetize travel expertise
- Need reliable platform for trading bookings
- Require multiple bookings management

## 3.3 Corporate User

**Profile**: Jennifer, 45, Corporate Travel Manager  
**Goals**:

- Manage employee bookings efficiently
- Reduce travel costs
- Centralized billing and reporting

**Pain Points**:

- Multiple booking platforms create complexity
- Limited control over employee bookings
- Need better reporting and analytics

## 3.4 Hotel Partner

**Profile**: David, 52, Hotel Revenue Manager  
**Goals**:

- Maximize occupancy rates
- Sell bulk inventory
- Reduce OTA dependency

**Pain Points**:

- High commission rates on OTAs (15-25%)
- Difficulty managing excess inventory
- Limited direct booking channels

## 3.5 Administrator

**Profile**: Admin Team  
**Goals**:

- Manage platform operations
- Monitor transactions
- Resolve disputes
- Analyze performance
